Protein–Protein Interaction (PPI) Prediction Methods Summary

Chapter 7 References & External Resources

This chapter provides a consolidated list of resources useful for protein–protein interaction (PPI) research, including databases, benchmark datasets, computational tools, and recommended literature.
These resources support the methodologies described in Chapters 1–6.

7.1 Major PPI Databases

These databases curate experimentally validated or computationally predicted PPIs.

General PPI Databases

Database Description Link
STRING Integrated PPI database combining experiments, text-mining, co-expression, and genomics https://string-db.org
BioGRID Curated experimental PPI database https://thebiogrid.org
IntAct EBI repository for experimentally validated PPIs https://www.ebi.ac.uk/intact
DIP Database of Interaction Proteins http://dip.doe-mbi.ucla.edu
HINT High-quality interactomes across species http://hint.yulab.org
MINT Molecular Interaction database https://mint.bio.uniroma1.it
HuRI Human Reference Interactome (Y2H-based) https://interactome.baderlab.org

Specialized PPI and Interaction Databases

Database Focus Link
STRING Viruses Viral-host PPIs https://viruses.string-db.org
CORUM Protein complexes in human/mouse https://mips.helmholtz-muenchen.de/corum
3did Domain–domain interactions https://3did.irbbarcelona.org
iRefWeb Integrated interaction resource http://irefindex.org

7.2 Structural Databases for PPI Analysis

Database Description Link
PDB Worldwide Protein Data Bank https://www.rcsb.org
PDBePISA Interface & assembly analysis https://www.ebi.ac.uk/pdbe/pisa
HSSP Homology-derived structures https://swift.cmbi.umcn.nl/gv/hssp
IBIS PPI structural templates https://www.ncbi.nlm.nih.gov/Structure/ibis/ibis.cgi

7.3 Benchmark Datasets for PPI Prediction

Benchmark datasets are essential for evaluating PPI models, docking pipelines, and multimer prediction.

Docking Benchmarks

Dataset Description Link
Docking Benchmark v5.0 (DB5) Gold-standard dataset for protein docking https://zlab.umassmed.edu/benchmark
CAPRI Community-wide docking assessment https://www.ebi.ac.uk/pdbe/complex-pred/capri

Deep Learning Benchmarks

Dataset Description Link
DIPS 42,000 PPI complexes for ML training https://github.com/phillipburgess/DIPS
SKEMPI ΔΔG mutation effects on PPIs https://life.bsc.es/pid/skempi
STRING-ML / BioGRID-ML ML-ready PPI datasets from STRING/BioGRID (various GitHub repos)

Sequence-Based Prediction Benchmarks

Dataset Focus Link
HopPPI Sequence-based PPI benchmarks https://github.com/caoyx1999/HopPPI
HHRI Dataset Human–human PPI benchmark widely used in DL PPI papers
